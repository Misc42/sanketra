Sanketra

Your phone becomes your mic, trackpad, and pointer.

Windows

Download and double-click install.bat to run it.

If you see a blue SmartScreen warning, click More info then Run anyway. The installer only installs Python (if missing) and clones the Sanketra server.
macOS

Open Terminal (Cmd+Space, type "Terminal") and paste:

curl -fsSL https://raw.githubusercontent.com/Misc42/sanketra-server/master/install.sh | bash

Or download install.command and double-click it.

If macOS blocks the file, right-click it and choose Open, then click Open again in the dialog.
Linux

Open a terminal and run:

curl -fsSL https://raw.githubusercontent.com/Misc42/sanketra-server/master/install.sh | bash
